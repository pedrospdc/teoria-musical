##  Exercises 6.7 Exercícios Práticos

### 1.

Analyze the tríade types (M, m, \\(\left.\text{}{+}\right.\\), \\(\left.\text{}^{\circ}{}\right.\\)) using lead-sheet symbols. Sus2 and sus4 acordes are also included. 

![](external/images/unit1/PRAC-EXER-Tríade-ID.svg)

Answer.

1\. \\(\left.\text{A}^♯{}^{\circ}{}\right.\\), 2. \\(\text{F}\\)sus4, 3. \\(\left.\text{B}^♭{}{+}\right.\\), 4. \\(\left.\text{D}^♭{}\right.\\), 5. \\(\text{E}\\)sus2 

### 2.

Write the specified tríades and sus acordes. 

![](external/images/unit1/PRAC-EXER-Tríade-writing.svg)

Answer.

1\. \\(\text{B}\\)\\(\text{D}\\)\\(\text{F}^♯\\), 2. \\(\text{E}\\)\\(\text{G}^♯\\)\\(\text{B}^♯\\), 3. \\(\text{A}^♭\\)\\(\text{B}^♭\\)\\(\text{E}^♭\\), 4. \\(\text{C}^♯\\)\\(\text{E}^♯\\)\\(\text{G}^♯\\), 5. \\(\text{D}\\)\\(\text{G}\\)\\(\text{A}\\), 6. \\(\text{F}\\)\\(\text{A}^♭\\)\\(\text{C}^♭\\)

### 3.

Analyze the following inverted tríades using slash notation. 

![](external/images/unit1/PRAC-EXER-tríade-inverted-ID.svg)

Answer.

1\. \\(\left.\text{D}\text{m}\middle/\text{F}\right.\\), 2. \\(\left.\text{G}\text{m}\middle/\text{D}\right.\\), 3. \\(\left.\text{A}^♯{}^{\circ}{}\middle/\text{C}^{♯}\right.\\), 4. \\(\left.\text{F}\middle/\text{A}\right.\\), 5. \\(\left.\text{B}^♭{}\middle/\text{F}\right.\\), 6. \\(\left.\text{C}\text{m}\middle/\text{G}\right.\\)

### 4.

Write the specified inverted tríades. 

![](external/images/unit1/PRAC-EXER-Tríade-Inverted-Writing.svg)

Answer.

(lowest to highest) 1. \\(\text{D}\\)\\(\text{G}\\)\\(\text{B}\\), 2. \\(\text{D}\\)\\(\text{F}^♯\\)\\(\text{B}\\), 3. \\(\text{F}\\)\\(\text{A}^♭\\)\\(\text{D}^♭\\), 4. \\(\text{B}^♭\\)\\(\text{E}\\)\\(\text{G}\\)

### 5.

Correct the misspelled tríades. Label your corrected spelling with lead-sheet notation. All of the examples are in root position (the lowest nota is the root). 

![](external/images/unit1/PRAC-EXER-Tríades-misspelled.svg)

Answer.

1\. \\(\text{F}^♯\\)\\(\text{A}^♯\\)\\(\text{C}^♯\\), 2. \\(\text{G}\\)\\(\text{B}^♭\\)\\(\text{D}^♭\\), 3. \\(\text{A}^♭\\)\\(\text{C}^♭\\)\\(\text{E}^♭\\)
